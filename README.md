# Two-Wheeler-Business-Intelligence-Report
Bike Sales Analysis project using SQL and Tableau to analyze resale trends, brand performance, year-wise sales, with version control managed using GitHub.

# Project Overview

The goal of this project is to perform end-to-end data analysis on bike sales data. SQL is used for querying and aggregating data, while Tableau is used to create interactive dashboards for better business understanding.

# Business Objectives

Analyze total resale value by bike brand

Identify each brandâ€™s contribution to overall resale revenue

Compare sales across different years of manufacture

Visualize insights to support data-driven decisions

# Tools and Technologies Used

SQL for data extraction, aggregation, and analysis

Tableau for data visualization and dashboard creation

Excel for initial data review and validation

Git and GitHub for version control and project management using Visual Studio Code

# Key Analysis Performed

Brand-wise total resale value analysis

Percentage contribution of each brand to total resale value

Year-wise sales comparison

Brand-wise and year-wise aggregation of resale prices

# Data Visualization

Interactive Tableau dashboards were created to display:

Brand-wise resale contribution

Year-wise sales trends

Total resale value summary

Filters for brand and year for dynamic analysis

# Version Control

Git commands were executed using Visual Studio Code to track SQL scripts and Tableau files, commit changes, and push the complete project to GitHub.

# Conclusion

This project demonstrates practical skills in SQL, Tableau, and GitHub while showcasing the ability to transform raw sales data into meaningful business insights
